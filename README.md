# CornPass Movies UI Kit – SwiftUI

A SwiftUI UI showcase project inspired by the Figma design:

[CornPass • Movies Booking UI Kit](https://www.figma.com/design/FIIGmQGZneGwadmIDnYxje/CornPass-%E2%80%A2-Movies-Booing-UI-Kit--Community-?node-id=2078-1042&t=ACVBTtxh7aaiPbCG-1)

This project is built purely for **UI development and SwiftUI practice**.  
It contains only static/mock data and focuses entirely on creating modern, animated, reusable SwiftUI interfaces.

---

# Overview

CornPass is a movie booking UI concept showcasing:

- Modern SwiftUI layouts
- Smooth animations
- Reusable components
- Dark themed cinema experience
- Clean design system
- Interactive UI flows

This project does **not** include:

- Backend integration
- APIs
- Database
- Authentication
- Payment processing

Everything is powered using local mock data for UI demonstration purposes only.
